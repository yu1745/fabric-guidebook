# Fabric Guidebook

This is the built-in guide used to validate the generic guidebook loader.

## Resource packs

- Define guides in `assets/<namespace>/guidebook_guides/<guide_id>.json`
- Put Markdown pages under the guide folder
- Open pages with resource links like [Second Page](second.md)
- Render SNBT structures with [3D Scene](scene.md)
